# Contributor Code of Conduct

1. We welcome all those who come in peace to support OpenPNGStudio!
2. Responsible behavior and mutual respect is expected from everyone on this project.
3. If chaos erupts from forgetting the above we will enforce our way to a more harmonious and productive environment.
4. We do not care about your skin color, politics, sexuality, or anything else. We won't bring up your personal life choices and nor should you.
5. Remember, this is a place every that shares a love of OpenPNGStudio. So please play nice.

This code of conduct is adapted from the [Devuan Code of Conduct](https://dev1galaxy.org/viewtopic.php?pid=54592#p54592).
